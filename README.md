# 🚀 API de Clientes (Do Zero)

Este projeto é uma API de gerenciamento de clientes desenvolvida do zero com Python, com o objetivo de praticar conceitos de back-end como rotas, CRUD, organização de código e integração com banco de dados.

## 📌 Funcionalidades

- ✅ Criar clientes  
- 📄 Listar clientes  
- 🔍 Buscar cliente por ID  
- ✏️ Atualizar dados de clientes  
- ❌ Deletar clientes  

## 🛠️ Tecnologias utilizadas

- Python 🐍  
- FastAPI ⚡  
- Uvicorn  
- Pydantic  
- SQLite

## COMANDO DE INICIALIZAÇÃO
 - uvicorn app.main:app --reload
